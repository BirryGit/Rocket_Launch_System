# Rocket Launch System
## What this is
This is a system built to launch mid-powered rockets. The system includes an electronic launch box and a rail for guidance. This project is as simple as possible without compromising look or features.
## Why I built it
I am very nearly finished with my mid-powered rocket build. The rocket kit is called Forte. The problem is launching. I need an electric launch system to set off the electric igniter fuse, and I need a rail to guide the rocket until the fins can take over.
## How to launch
The rocket is put in the slot at the end of the rail, then the 30 ft wire is connected to the rocket's fuse. Once you check the continuity LED, you can flip all of the switches and press the launch buttons!
